# wemedia-smart-assistant

一个自毁体浏览器插件，代替作者做一些点击、评论的体力活

## 如何使用

需要在本地创建一个mysql database: chrome_extension_assistant。

需要先启动后端

```bash
cd server
go get github.com/codegangsta/gin
export GO111MODULE=on
go mod download
./dev.sh
```

直接在chrome浏览扩展中添加根目录为插件。插件名称为“自媒体智能助理”。
